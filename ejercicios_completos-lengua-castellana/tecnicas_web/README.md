## 📖 Descripción

**Sistema de Gestión de Aprendizaje (LMS)** completo para técnicas de lengua castellana. Plataforma educativa con:
- 📚 **Biblioteca de técnicas** (309+ recursos)
- 🎓 **Cursos y lecciones** interactivas
- 🏆 **Gamificación** con puntos e insignias
- 📝 **Evaluaciones** con múltiples tipos de preguntas
- 👥 **Gestión de usuarios** (estudiantes, profesores, administradores)
- 📊 **Seguimiento de progreso** en tiempo real

## ✨ Características Principales

### 📚 Contenido Educativo
- **309+ técnicas** de lengua castellana
- **14 categorías** organizadas temáticamente
- **Quizzes interactivos** HTML autónomos
- **Recursos descargables** y material de apoyo

### 🎓 Sistema de Cursos
- **Creación y gestión** de cursos
- **Lecciones estructuradas** con contenido asociado
- **Seguimiento de progreso** por estudiante
- **Inscripción automática** a cursos

### 🏆 Gamificación
- **Sistema de puntos** por actividades completadas
- **Insignias** por logros (bronce, plata, oro, platino, diamante)
- **Ranking** de usuarios
- **Racha de días** de actividad
- **Actividades recientes** registradas

### 📝 Evaluaciones
- **Múltiples tipos de preguntas:**
  - Opción Múltiple
  - Verdadero/Falso
  - Completar espacios
  - Preguntas Abiertas
- **Calificación automática** para preguntas objetivas
- **Resultados detallados** por evaluación
- **Historial de intentos**

### 👥 Gestión de Usuarios
- **Tres roles:** Administrador, Profesor, Estudiante
- **Perfiles personalizados** por rol
- **Panel de administración** completo
- **Permisos granulares** por rol
